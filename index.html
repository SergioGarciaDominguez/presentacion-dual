Índice

Solbyte

El principal proyecto en el que he participado ha sido un proyecto no real centrado en desarrollar desde cero una web en PHP, con diversas páginas y formularios que enlazan con una base de datos.

Las principales asignaturas con las que está relacionado es con programación, lenguajes de marcas, y bases de datos. Aunque también implica algo de Sitemas Infórmaticos y entornos de desarrollo.

Html

Sistemas

El sistema que he usado en la empresa es un ordenador portátil con procesador AMD y 4GB de RAM. El SO es un Windows 10 pro.

La red que usan es una red centralizada en estrella, donde un OC ocupa la posición central y todos los demás dependen de este.

En una ocasión también apliqué lo dado en sistemas informáticos para publicar contenido en un HOST con filezilla.

Entornos de desarrollo

Solbyte como empresa utiliza metodologías ágiles a la hora de desarrollar y está muy en contacto con el cliente, ya que requiere de su feedback para llevarlo a cabo adecuadamente.

En cuanto al lenguaje de desarrollo, Solbyte desarrolla en PHP prácticamente el 100% de los proyectos que destina a webs (backend). Lógicamente complementado con Javascript en frontend. Ambos lenguajes son interpretados, con un sistema de tipos dinámico y débil (no estricto).

Lenguajes de marcas

Como es obvio, he utilizado en el proyecto el lenguaje de marcas por excelencia, HTML5. Usando etiquetas novedosas y funcionales de este: header, footer, nav, section, etc.

He aprendido también a usar mejor las características de CSS3, para la apariencia. Selectores, propiedades, así como el uso de media-queries en la página principal, para lograr un diseño responsivo sin ayuda de frameworks.

Programación

La programación la he llevado a cabo usando PHP y Javascript. He usado una programación estructurada, no orientada a objetos; suficiente para un proyecto pequeño.

Cada web dispone de un controlador en PHP que recibe los datos por GET o POST (o ambos). Este controlador se encarga de conectar a la base de datos, en caso necesario, y de devolver la información necesaria de nuevo a la vista.

Bases de datos

La base de datos usada en este proyecto está basada en MYSQL, y he interactuado con ella a través de PHPmyAdmin.

Ha sido una base de datos sencilla en la que se relacionaba la entidad Tareas con Usuarios, relacionadas por una tabla intermedia, que recogía que Tareas eran realizadas por qué usuarios y en qué fecha (día).

Además hay una cuarta tabla independiente que simplemente recogía las opiniones de los usuarios de la página principal.
